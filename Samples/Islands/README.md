# WinAppSDK Islands Sample Apps

This directory contains samples that demonstrate how to use WinAppSDK islands in a different types of apps.
* The [cpp-win32-unpackaged](cpp-win32-unpackaged\README.md) directory shows how to use an island in a Win32 app.
* The [cs-winforms-unpackaged](cs-winforms-unpackaged\README.md) directory shows how to use an island in an unpackaged WinForms app.
